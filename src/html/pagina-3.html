Essa página está ruim, não está?

Elementos essenciais de um website moderno estão faltando ou estão errados aqui. Você consegue adivinhar o quê?

Nota: esta lista deve estar oculta a princípio.

  • Layout tão datado que tem idade para dirigir
  • GIFs e animações exageradas, inclindo aquela faixa lá em cima.
  • Comic Sans...
  • Falta de responsividade
  • Quebrou o padrão de puzzles que o próprio site estabeleceu nas páginas anteriores
  • Menu sem consistência de ordem e com iconografia/nomes não convencionais, simulando um site que não segue convenções

Esta página diz qual dos puzzles o usuário precisa resolver para passar a fase

Formulário com funcionalidade ruim

  • Labels com posições ambíguas
  • Campos obrigatórios mal indicados e sem explicação, sem seguir convenções
  • Falta de validação/regex nos campos de texto
  • Todos são inputs de texto, mesmo quando o mais adequado seria outro (data, dropdown...)
  • Auto foco no campo que não devia
  • Falta placeholder ou outras indicações para o formato de data
