Enigmas de conhecimentos do Web design

Você passará por enigmas, encontre a pista e digite a senha para avançar de etapa.

1 - Inicie o enigma :)

Cada etapa possuirá três ou dois enigmas. Cada enigma possui uma senha para avançar de etapa.

2 - Busque a senha...

Ao encontrar a resposta, digite no campo que estará disponível em cada enigma. Se tiver dificuldade, clique no botão de ajuda!

3 - Desvende todos os enigmas!

Para avançar, você deve digitar corretamente as senhas de cada enigma.
